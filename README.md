# HTML5 存档编辑器 (HTML5 Savegame editors)
一套使用 HTML5 技术制作的主机游戏存档编辑器集合。

当前可编辑下列游戏部分存档内容：
 * 《塞尔达传说 王国之泪》(The Legend of Zelda: Tears of the Kingdom，Switch)
 * 《塞尔达传说 旷野之息》(The Legend of Zelda: Breath of the Wild，Wii U / Switch)
 * 《任天堂明星大乱斗 特别版》(Super Smash Bros. Ultimate，Switch)
 * 《塞尔达无双》(Hyrule Warriors，Wii U)
 * 《光神话：帕露提娜之镜》(Kid Icarus: Uprising，3DS)
 * 《最终幻想探险者》(Final Fantasy Explorers，3DS)
 * 《马力欧卡丁车7》(Mario Kart 7，3DS)
 * 《超级卡比猎人队》(Super Kirby Clash，Switch)
 * 《卡比猎人队 豪华版》(Team Kirby Clash Deluxe，3DS)
 * 《卡比的吸入大作战》(Kirby's Blowout Blast，3DS)
 * 《立体绘图方块2》(Picross 3D Round 2，3DS)
 * 《寿司道 炫斗食侍》(Sushi Striker: The Way of Sushido，3DS)
 * 《宝可梦填字方块》(Pokémon Picross，3DS)
 * 《宝可梦消消乐》(Pokémon Shuffle，3DS)
 * 《节奏天国 混音》(Rhythm Paradise Megamix，3DS)
 * 《Mii广场》(StreetPass Mii Plaza，3DS)

## Electron 客户端运行
1. 安装依赖：在仓库根目录执行 `npm install`
2. 启动客户端：执行 `npm start`（默认直接打开《塞尔达传说 王国之泪》编辑器）
